Troubleshooting & Service Logs
Logs · Service health · Diagnostics · 2 hrs
▾
Task 15.1 — Exam Style
Instance web-01 shows status ERROR. Diagnose using service logs and CLI only. Fix the issue.
# Check fault message first
openstack server show web-01 -c fault -c status -c vm_state

# Check compute services health
openstack compute service list

# Nova API logs
sudo journalctl -u nova-api --since "30 minutes ago" -n 100

# Nova compute logs
sudo journalctl -u nova-compute --since "30 minutes ago" -n 100

# Nova scheduler logs
sudo journalctl -u nova-scheduler --since "30 minutes ago" -n 50

# Neutron service logs
sudo journalctl -u neutron-server --since "30 minutes ago" -n 50

# Glance logs
sudo journalctl -u glance-api --since "30 minutes ago" -n 50

# All OpenStack services status
openstack service list --long
Task 15.2 — Exam Style
Project dev-team has hit its instance quota. Show current usage. Increase instance quota to 20. Increase volume quota to 15. Verify both changes.
# Show current usage
openstack quota show --usage dev-team

# Increase quotas
openstack quota set dev-team --instances 20
openstack quota set dev-team --volumes 15

# Verify
openstack quota show dev-team | grep -E "instances|volumes"
Break Something — Fix It
Intentionally break your environment, then diagnose and fix without looking at notes:
# Break 1: Remove router subnet interface
openstack router remove subnet main-router app-subnet
# Now diagnose: why can't instances reach the internet?
# Fix: openstack router add subnet main-router app-subnet

# Break 2: Disable a network
openstack network set app-net --disable
# Now diagnose: why can't instances launch?
# Fix: openstack network set app-net --enable

# Break 3: Disassociate floating IP
openstack server remove floating ip web-01 $FIP
# Now diagnose: why is web-01 unreachable?
# Fix: openstack server add floating ip web-01 $FIP

CLI Speed Tricks — Screen, Output Formatting, Aliases
Time multipliers · Learn these once, use forever · 2 hrs
▾
The screen Command — Never Wait Idle
When --wait blocks for 60+ seconds, screen lets you work in another window simultaneously.
# Start screen
screen

# New window:       Ctrl+A then C
# Next window:      Ctrl+A then N
# Previous window:  Ctrl+A then P
# Jump to window N: Ctrl+A then 0/1/2
# Split screen:     Ctrl+A then S
# Move to pane:     Ctrl+A then Tab
# Detach:           Ctrl+A then D
# Reattach:         screen -r

# EXAM STRATEGY: Window 0 = long-running tasks
# Window 1 = quick tasks while waiting
Output Formatting — Extract IDs Instantly
# Get just the value of one field
openstack project show dev-team -f value -c id

# Capture into variable
PROJECT_ID=$(openstack project show dev-team -f value -c id)
NET_ID=$(openstack network show app-net -f value -c id)
FIP=$(openstack floating ip list -f value -c "Floating IP Address" | head -1)

# Show only specific columns
openstack server list -c Name -c Status -c Networks

# Filter by property
openstack server list --status ERROR
openstack server list --status ACTIVE

# Pipe to grep
openstack server list | grep web

# JSON output for scripting
openstack server show web-01 -f json | python3 -m json.tool
Exam-Start Aliases — Set These First Thing
# Add to ~/.bashrc OR run at exam start
alias os='openstack'
alias osl='openstack server list'
alias osn='openstack network list'
alias osv='openstack volume list'
alias osi='openstack image list'

function getid() {
  openstack $1 show $2 -f value -c id
}
# Usage: getid server web-01
# Usage: getid network app-net
# Usage: getid project dev-team

source ~/.bashrc

Common Weak Zone 1 — Swift ACL Syntax
# Drill until automatic — no looking up
openstack container create test-acl-container

# Public read
openstack container set test-acl-container --property "X-Container-Read=.r:*,.rlistings"

# Private again
openstack container unset test-acl-container --property "X-Container-Read"

# Auto-delete object
openstack object create test-acl-container /etc/hostname --name test.txt
openstack object set test-acl-container test.txt --property "X-Delete-After=3600"

# Verify everything
openstack container show test-acl-container
openstack object show test-acl-container test.txt
