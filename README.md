A complete **21-day study guide** for the [Certified OpenStack Administrator (COA)](https://openstack.org/coa) exam — built by Horus Sielatshom.

| File | Content |

## Exam Overview

- **Format:** Hands-on, live OpenStack environment (no multiple choice)
- **Duration:** 3 hours
- **Pass mark:** ~70%
- **Tools allowed:** CLI terminal + Horizon + docs.openstack.org only

## Domain Weights

| Domain | Weight |
|--------|--------|
| Compute (Nova) | 35% |
| Networking (Neutron) | 30% |
| Identity (Keystone) | 15% |
| Block Storage (Cinder) | 10% |
| Images (Glance) | 5% |
| Object Storage (Swift) | 5% |

## Usage

Open `COA_00_Master_Index.html` in your browser. All files are cross-linked — work through them in order, Days 1–21. Every task is exam-format with full CLI solutions.
